# Portfolio Website

A modern, responsive portfolio website built to showcase projects, skills, and professional experience.

## Tech Stack

- **JavaScript** - Main programming language
- **React** - UI library for building the interface
- **Vite** - Fast build tool and development server
- **Tailwind CSS** - Utility-first CSS framework for styling
- **Framer Motion** - Animation library for smooth transitions
- **React Icons** - Icon library for social media and UI icons

## Features

- Responsive design for mobile and desktop
- Smooth scroll animations on all sections
- Project showcase with images and descriptions
- Downloadable resume
- Social media links (LinkedIn, GitHub)
- Contact section

## Installation

```bash
# Clone the repository
git clone https://github.com/Cj5822/portfolio-website.git
cd portfolio-website

# Install dependencies
npm install
```

## Run the App

```bash
# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
portfolio-website/
├── public/          # Static assets (images, resume)
├── src/
│   ├── components/  # React components (Hero, NavBar, Projects, etc.)
│   ├── App.jsx      # Main app component
│   └── main.jsx     # Entry point
└── package.json
```

## Deployment

Build the project and deploy the `dist` folder to your hosting provider:

```bash
npm run build
```
